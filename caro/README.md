# Hướng dẫn test Cờ Caro trên 1 máy (local)

1. Yêu cầu:
   - Python 3.x đã cài.
   - Thư viện chuẩn (không cần thư viện ngoài).

2. Các bước:
   - Mở terminal, chuyển tới thư mục chứa project (file server.py, client.py).
   - Chạy server:
     ```
     python server.py
     ```
   - Mở 2 terminal khác hoặc chạy 2 lần ứng dụng client GUI:
     ```
     python client.py
     ```
     Mỗi cửa sổ client sẽ là 1 người chơi; nhấn "Tìm trận" để ghép đôi.
   - Nếu muốn chạy client GUI nhiều bản, bạn có thể double-click hoặc mở nhiều terminal.
